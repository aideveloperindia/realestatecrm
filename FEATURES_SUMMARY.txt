4. WHEN A NEW PROPERTY IS ONBOARDED, WHERE ARE SUGGESTIONS FOR SUITABLE CUSTOMERS?
--------------------------------------------------------------------------------
✅ FEATURE: Property → Customer Matching

Location: Properties → [View Property] → "Suggested Customers" section

Steps:
1. Add a new property (Properties → Add Property)
2. After creating, click "View" on that property
3. Scroll down to "Suggested Customers (Top Matches)" section
4. See top 10 matching customers with:
   - Customer name, phone, email
   - Budget range
   - Match Score (0-100)
   - Price Score, Location Score, Type Match percentages
   - Match Reasons (why it matches)
   - "Send WhatsApp" button for each customer

The matching algorithm considers:
- Property price vs customer budget
- Property location vs customer preferred location
- Property type vs customer preferred types


5. WHEN A NEW CUSTOMER IS ONBOARDED, WHERE ARE SUGGESTIONS FOR SUITABLE PROPERTIES?
--------------------------------------------------------------------------------
✅ FEATURE: Customer → Property Matching

Location: Customers → [View Customer] → "Property Matches" section

Steps:
1. Add a new customer (Customers → Add Customer)
2. After creating, click "View" on that customer
3. Scroll down to "Property Matches" section
4. See top 10 matching properties with:
   - Property title, type, price, location
   - Match Score (0-100)
   - Price Score, Location Score, Type Match percentages
   - Match Reasons (why it matches)
   - "Send WhatsApp" button for each property

The matching algorithm considers:
- Customer budget vs property price
- Customer preferred location vs property location
- Customer preferred types vs property type
